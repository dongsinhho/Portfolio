## Installation

To run this project locally, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/dongsinhho/Portfolio.git
    ```
2. Start API:
    ```bash
    cd Portfolio/server
    dotnet run
    ```
3. Start React App
    ```bash
    cd Portfolio/client
    npm start
    ```
